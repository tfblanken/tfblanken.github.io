**2019**

Media coverage on the paper (in different languages) can be found <a href="https://pure.knaw.nl/portal/en/clippings/meer-helderheid-in-slapeloosheid(144caf18-50c7-48b1-a7a5-0ba90281e831).html">here</a>.
